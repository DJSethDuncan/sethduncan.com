Site is auto-deployed to Heroku on commits to main branch.

**Why:** Heroku pipeline connected to GitHub main branch for continuous deployment.

**How to apply:** No manual deploy steps needed — merging/pushing to main triggers a deploy automatically. When advising on changes, note that they'll go live as soon as they land on main.
